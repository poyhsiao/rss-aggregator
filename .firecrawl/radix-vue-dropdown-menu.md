## Features [​](https://www.radix-vue.com/components/dropdown-menu\#features)

- Can be controlled or uncontrolled.
- Supports submenus with configurable reading direction.
- Supports items, labels, groups of items.
- Supports checkable items (single or multiple) with optional indeterminate state.
- Supports modal and non-modal modes.
- Customize side, alignment, offsets, collision handling.
- Optionally render a pointing arrow.
- Focus is fully managed.
- Full keyboard navigation.
- Typeahead support.
- Dismissing and layering behavior is highly customizable.

| `sticky` |  | `'partial' | 'always'`<br>The sticky behavior on the align axis. `partial` will keep the<br>content in the boundary as long as the trigger is at least partially<br>in the boundary whilst "always" will keep the content in the boundary<br>regardless. |
| `updatePositionStrategy` |  | `'always' | 'optimized'`<br>Strategy to update the position of the floating element on every animation frame. |

| Emit | Payload |
| --- | --- |
| `closeAutoFocus` | `[event: Event]`<br>Event handler called when auto-focusing on close.<br>Can be prevented. |
| `entryFocus` | `[event: Event]` |
| `escapeKeyDown` | `[event: KeyboardEvent]`<br>Event handler called when the escape key is down.<br>Can be prevented. |
| `focusOutside` | `[event: FocusOutsideEvent]`<br>Event handler called when the focus moves outside of the `DismissableLayer`.<br>Can be prevented. |
| `interactOutside` | `[event: PointerDownOutsideEvent | FocusOutsideEvent]`<br>Event handler called when an interaction happens outside the `DismissableLayer`.<br>Specifically, when a `pointerdown` event happens outside or focus moves outside of it.<br>Can be prevented. |
| `openAutoFocus` | `[event: Event]`<br>Event handler called when auto-focusing on open.<br>Can be prevented. |
| `pointerDownOutside` | `[event: PointerDownOutsideEvent]`<br>Event handler called when the a `pointerdown` event happens outside of the `DismissableLayer`.<br>Can be prevented. |

| Data Attribute | Value |
| --- | --- |
| `[data-state]` | `"open" | "closed"` |
| `[data-side]` | `"left" | "right" | "bottom" | "top"` |
| `[data-align]` | `"start" | "end" | "center"` |
| `[data-orientation]` | `"vertical" | "horizontal"` |

### Constrain the content/sub-content size [​](https://www.radix-vue.com/components/dropdown-menu\#constrain-the-content-sub-content-size)

      </DropdownMenuContent>

## Accessibility [​](https://www.radix-vue.com/components/dropdown-menu\#accessibility)

Adheres to the [Menu Button WAI-ARIA design pattern](https://www.w3.org/WAI/ARIA/apg/patterns/menubutton) and uses [roving tabindex](https://www.w3.org/WAI/ARIA/apg/patterns/kbd_roving_tabindex) to manage focus movement among menu items.

### Keyboard Interactions [​](https://www.radix-vue.com/components/dropdown-menu\#keyboard-interactions)

| Key | Description |
| --- | --- |
| `Space` | When focus is on `DropdownMenuTrigger`, opens the dropdown<br>menu and focuses the first item.<br> <br>When focus is on an item, activates the focused item. |
| `Enter` | When focus is on `DropdownMenuTrigger`, opens the dropdown<br>menu and focuses the first item.<br> <br>When focus is on an item, activates the focused item. |
| `ArrowDown` | When focus is on `DropdownMenuTrigger`, opens the dropdown<br>menu.<br> <br>When focus is on an item, moves focus to the next item. |
| `ArrowUp` | When focus is on an item, moves focus to the previous item. |
| `ArrowRight`  `ArrowLeft` | When focus is on `DropdownMenuSubTrigger`, opens or closes<br>the submenu depending on reading direction. |
| `Esc` | Closes the dropdown menu and moves focus to{' '}<br>`DropdownMenuTrigger`. |
