# Radix Vue

Vue Port of Radix UI

Unstyled, accessible components for building high‑quality design systems and web apps in Vue.

[Get Started](https://www.radix-vue.com/overview/getting-started.html)

[View on GitHub](https://github.com/unovue/radix-vue)

![Radix-Vue](https://www.radix-vue.com/logo.svg)

## Vue, Nuxt supported.

Build for Vue framework, which compatible with any meta framework build on top of Vue.

🚀

## Save time. Ship faster.

Building on top Radix components will save you time and money, so you can ship a better product faster.

⌨️

## Accessibility out of the box.

WAI-ARIA compliant. Support Keyboard navigation & Focus management.

🧑🏻‍💻

## Developer Experience First.

Unstyled, easily customizable and great for building design system and web apps.
